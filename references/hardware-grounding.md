# 硬件落地约束

## 作用

本文件用于把展陈硬件判断规则转成生图约束，避免装置图只在文字描述上成立，但设备距离、安装方式、观看距离、维护空间或交互范围不成立。

本文件只保留已经抽象成规则的硬件类别、空间关系和使用限制。

不要把未授权材料里的设备照片、品牌、型号或案例造型复制进图里。只使用这里沉淀后的通用硬件判断。

## 生成前硬件检查

涉及投影、屏幕、LED、触控、感应、声音、气味、服务器/控制设备时，生成前必须补全：

- 设备是什么：投影机、短焦投影、背投、LCD 商显、拼接屏、LED 点阵、透明屏、地踩 LED、触摸屏、红外框、雷达、Kinect、编码器、音响、闻香口、气味模块、控制主机等。
- 设备放在哪里：吊装、壁挂、落地、桌内、展台内、天花、墙后、屏幕后、轨道、支架、机柜。
- 设备和目标面的关系：投射距离、观看距离、识别距离、声场覆盖、屏幕高度、操作距离。
- 人会不会挡住：投影光路、感应范围、音响声场、屏幕视线是否被人或展台遮挡。
- 维护是否成立：厚屏、DLP 箱体、LED 箱体、投影吊装、机柜、散热和检修空间是否需要画出或暗示。
- 空间光环境是否匹配：投影怕亮、LED/商显适合明亮空间、OLED/透明屏怕强反光或成本限制。
- 装置形态是否轻便：能否做成独立展台、立式小装置、手摇/旋钮/把手机构、移动底座或桌面装置，而不是默认顶天立地轨道、吊架或大型结构。

## 装置形态优先级

展览中的互动装置通常要服从空间秩序。除非用户明确要求大型结构、沉浸空间、吊装系统或建筑级装置，默认优先选择轻便、独立、可维护、低侵入的设备形态。

优先级：

1. 独立轻量装置：落地立式展台、带底座的小型设备、桌面设备、手持/半手持道具、可移动推车、低位儿童装置。
2. 家具化装置：互动桌、斜面操作台、展示柜旁设备、墙前独立设备、与展柜/展墙整合的局部装置。
3. 局部安装装置：短轨道、小支臂、壁挂小模块、天花小点位、墙前轻型支架。
4. 大型结构：顶天立地轨道、龙门架、全高立柱、天花吊架、大型桁架、建筑构件级装置。

默认不要直接使用第 4 类。只有在以下情况才使用大型结构：

- 用户明确指定全高轨道、吊装、沉浸空间、建筑构件或大型机械。
- 功能必须跨越很大高度或面积，且轻量装置无法完成。
- 需要承重、安全隔离、群体观看或大型投影融合。
- 空间本身就是黑场、剧场、环幕、沉浸厅或设备机房型展项。

画法规则：

- 优先把交互设备做成独立可识别的展陈产品，而不是把墙、天花和地面都变成设备结构。
- 设备可以有底座、斜面小屏、手摇柄、旋钮、把手、转轴、灯头、支架和维护门。
- 轻量装置仍然要满足工作距离、可达范围和稳定性：有底座、有支撑、有线缆或维护口暗示。
- 不要为了表现“可上下移动”就默认画成顶天立地滑轨；优先考虑手摇、旋钮、齿轮、短滑轨、转轴、倾角调节、台座升降或灯头俯仰。
- 不要破坏空间秩序：避免过高、过粗、过多的立柱、轨道、桁架、吊臂遮挡展墙或主体内容。

## 投影类

投影不是一个“发光面”，必须有投射方式。

投影判断的第一优先级不是“有没有墙面”和“有没有距离”，而是光学轴线是否成立。先定投影源的镜头朝向，再沿镜头方向放置目标面；不要先在右侧、后方或空白处补一个墙面结果，再强行把光束连过去。

投影判断的第二优先级是目标介质是否和创意匹配。不要把“独立幕布/右侧屏幕”当成投影的默认结果。投影的价值通常在于让内容贴到某个空间、物体、模型、表面或半透明介质上，而不是把所有互动都变成“左边操作装置 + 右边显示屏”。

生成前先决定投影类型。不要用“有光束”笼统代替判断：

- 普通正投：需要可见投射距离，人物不能站进主光路。
- 短焦/超短焦：可以靠近目标面，但要看出短焦机身、贴墙/桌面支架、倾斜出光方向和落点。
- 背投：目标面后方需要设备空间，画面要暗示背后设备区、半透明幕或墙后投影。
- 桌面/模型投影：投影内容要贴合桌面、沙盘或模型形态，不能像竖屏 UI 贴图。
- 隐藏投影源：即使设备藏起来，也要用顶部小点位、墙面开口、设备轮廓或标注说明来源。
- 投影融合：只适合大尺度或异形融合，需要多源、暗场和维护条件，不作为普通互动默认方案。

### 目标介质选择

生图前先从创意内容选择目标介质，而不是默认选择墙或独立幕布：

- 物体/展品表面：适合标本、器物、模型局部、机械结构、胶片片段、可被照亮或放大的实物。
- 桌面/沙盘/地形模型：适合地图、路线、生态层、城市肌理、流域关系、网络关系。
- 装置本体表面：适合地球仪、转盘、透明筒、灯头、操作台、机械外壳上的局部反馈。
- 半透明屏/纱幕/膜：适合漂浮层、复原层、空间剖面，但必须看出介质存在。
- 地面/台阶/路径：适合脚下触发、行走路径、方向和范围，不适合承载长文本。
- 墙面/独立幕布：适合群体观看、较大画面、叙事片段或需要远距离阅读的信息；不是默认选项。
- 背投/内嵌投影：适合小型独立装置或台座内部反馈，可减少外部光路和右侧幕布同质化。

如果目标内容本来更适合屏幕、灯箱、透明显示、机械开合或实体模型变化，就不要为了保留“投影”而硬加外部幕布。普通小型互动装置优先考虑把反馈嵌入装置本体、模型表面或近距离介质中。

### 光学轴线门槛

生图前必须用一句话确定：

```text
投影源位置：{在哪里}；镜头朝向：{朝哪里}；目标面：{什么介质}；目标面位置：{在镜头正前方/斜前方/下方/背后}；投射距离：{大概多远}。
```

如果这句话说不清，不要进入生图；先改装置形态、改构图角度，或换成屏幕/灯箱/实体反馈。

画面里必须满足：

- 镜头朝向、光锥中心线、目标面位置三者共线或近似共线。
- 每个投影源只能有一个有效出光口。先决定是前置镜头、顶部反射窗、出光槽、墙面开口还是隐藏孔位在出光；光束必须从这个口径出发。不要让同一台设备的前镜头和顶部孔同时发光，除非明确是多投影融合且每个光路都独立成立。
- 目标面必须位于镜头前方的光锥内，不能因为画面右侧有空墙就把投影内容放到右墙。
- 目标面要有合理介质：墙面、幕布、桌面、模型、半透明屏、纱幕、地面、物体表面或背投屏。空中悬浮画面不算投影目标。
- 目标面的朝向要能接住光。墙面/幕布通常面对镜头；桌面/模型投影要让光从上方或斜上方落下；背投要暗示屏幕后方设备空间。
- 如果当前构图看不清镜头朝向，优先改成侧视或 3/4 轴测，让“投影源 -> 光锥 -> 目标面”成为一条直观链路。
- 不要把“右侧墙面投影”当成默认答案。墙面投影只有在镜头确实朝向墙面、人物不挡光、空间能暗下来的情况下才使用。
- 不要为了光轴清楚而固定使用侧视“左装置 + 右幕布”。只有当外部幕布是最适合内容阅读的介质时才这样画；否则优先选择桌面、模型、物体、装置本体、地面、半透明介质或背投。

### 常见方式

- 正投：投影机和观众在目标面同侧，容易被人挡光；画面里要避免人物站进主光路。
- 背投：投影机在屏幕后方，需要屏幕后空间；画面里可用半透明幕、背后设备区或墙后空间说明。
- 短焦/超短焦：设备可以靠近墙面或桌面，但必须有明显短焦机身、壁挂支架、贴墙支臂或倾斜光路，不能画成普通投影机贴着墙。
- 投影融合：适合大画面、异形面或沉浸空间，但需要多台投影机、融合边界、吊装/维护和可控光。
- 膜/纱幕/透明介质投影：需要看出介质存在，例如膜、纱幕、玻璃、雾幕或半透明面。

### 画法规则

- 必须画清：投影源 -> 光路 -> 目标面。
- 必须判断：人是否会挡光，空间是否能暗下来，投射距离是否成立。
- 必须分清：投影光路和交互联动箭头。投影光路只能从投影源出发，不能从手、按钮、道具或识别仓出发。
- 必须表现方向：设备出光面、光锥方向和落点方向要一致，不能出现投影机朝错方向却把内容投到另一侧。
- 必须表现口径一致：如果设备画了前镜头，光束就从前镜头出；如果光束从顶部反射窗或出光槽出，前镜头不能再发光，必要时把它画成散热孔、传感器或取消。
- 必须让目标面沿投影机镜头方向布置。不要让投影机朝左/朝下，却把结果画在右侧墙面；也不要让镜头朝观众，却把内容投到侧墙。
- 如果投影目标是桌面、沙盘、模型或墙面，要让内容贴合目标形态，不要像普通屏幕贴图。
- 如果投影目标是装置本体、物体或模型，投影内容要随表面弧度、厚度、材质或结构变化，不要变成旁边外接屏。
- 明亮开放空间不优先使用普通投影。若必须使用，要画出局部暗区、高亮短焦、遮光或可控光。
- 投影机不要无理由贴在目标面上。只有短焦/超短焦/嵌入式设备可以近距离，但仍要表现支架、出光方向和落点。
- 如果工作距离不足，优先改为短焦/超短焦、背投、内嵌屏、局部灯箱或实体反馈；不要硬画普通投影。

## LCD 商显 / 拼接屏

LCD 商显和拼接屏适合明亮空间、稳定播放、墙面信息、互动屏和展墙内容。

画法规则：

- 商显有厚度、边框和安装方式，不是无限薄发光面。
- 拼接屏有拼缝，即使很窄也要用轻微分割线暗示；不要画成无缝整屏。
- 大屏要考虑观看距离和屏幕高度。儿童展项不要把互动触摸点放到成人高度。
- 主反馈屏要考虑观看视角：细节内容、对比结果、地图、照片或文字反馈不能放在膝盖以下、踢脚线高度、展台底部正面或被操作台边缘遮挡的位置。
- 低位屏只适合儿童低位观看、地面/脚下反馈、状态灯、缩略提示或短暂确认；如果要承载主要结果，必须抬高、前倾或转成斜面/立面屏，让当前参与者自然看见。
- 明亮空间优先 LCD/LED/灯箱/实体反馈，而不是低亮度投影。
- 如果屏幕用于互动，人物手部或站位必须能触达或进入识别范围。

## LED 点阵 / 透明屏 / 柔性屏 / 地踩 LED

LED 适合高亮、开放空间、大尺度显示、异形和可踩地面，但必须考虑点间距、观看距离、维护和结构。

画法规则：

- LED 大屏可用模块格、轻微像素感、箱体厚度或结构骨架暗示。
- 观看距离要和点间距匹配。近距离儿童互动不应画成粗颗粒大屏，除非内容是低精度图形或灯光状态。
- 透明 LED 要和橱窗、玻璃、空间透视或后方展品关系成立；不要画成普通透明魔法屏。
- 柔性 LED 可用于弧形、柱面、曲面，但要暗示结构支撑。
- 地踩 LED 适合脚下触发、路径、状态、游戏式反馈；不适合承载长文本和细密地图。
- LED 异形加工成本高，图中不要为了炫酷任意堆异形屏，除非创意确实需要。

## 触控与感应

触控和感应不是抽象光圈，必须有触发对象、识别范围和安装位置。

### 常见类型

- 电容触摸：适合屏幕、桌面、近距离手指操作。
- 红外触摸框：适合较大屏幕或边框式交互，需暗示屏幕边框/红外框。
- 激光雷达/雷达：适合空间定位、靠近、手势、多人位置识别，需画检测扇区或扫描范围。
- Kinect/深度摄像头：适合人体骨骼、动作捕捉、站位互动，需有面对人的安装点和视场。
- 各类传感器：压力、距离、光电、磁吸、RFID、编码器、旋钮、拉杆、手柄等。
- 编码传感器：适合机械把手、曲柄、滑杆、轨道、转盘，把物理动作转换成数值或阶段反馈。

### 画法规则

- 感应范围必须覆盖人物动作或道具位置。
- 摄像头/Kinect/雷达不能背对参与者，也不能被墙、屏幕或展台挡住。
- 触摸屏要和手指/手掌接触；空间感应要画检测区而不是强行接触。
- 编码器、滑杆、拉杆、旋钮必须让人物能自然握持，不能超出人体可达范围。
- 不要同时堆多种传感器，除非玩法确实需要多输入。

## 声音系统

声音设备要考虑覆盖范围、方向、环境噪声和建声条件。

画法规则：

- 定向声、局部声场、耳机、听筒、座椅声源、顶部声源要有明确听音位置。
- 多声道或环绕声需要人物站位/座位和声源分布，不要只画几个音响盒。
- 展厅如果人流嘈杂，不适合只靠远距离小声源传达核心信息。
- 声音内容要能被画面理解，例如口述史、环境音、警示、提示、方位声场。

## 气味 / 嗅觉模块

气味设备要考虑释放范围、嗅闻距离、换样维护和通风条件。

画法规则：

- 气味模块必须有明确安装位置，例如桌面边缘、样本托盘、展柜旁小盒、床头模块、手持闻香器或设备面板。
- 气味释放范围应局部、短距离、朝向参与者嗅闻动作；不要画成充满空间的大雾、舞美烟或蓝色光束。
- 如果需要多种气味，优先用 2-4 个样本、胶囊、替换仓或选择位表达，不要画成一排说明牌。
- 气味设备不能贴到人物身体上；若是手持闻香器，必须有可握持外壳、出气口和明确手持关系。
- 气味通常需要辅助屏、灯环、样本高亮或短标签说明对应内容，但辅助反馈不能变成完整说明面板。

## 控制与机柜

控制主机、服务器、功放、媒体服务器、交换机等通常不应成为装置图主角。

画法规则：

- 只有当用户要求制作说明、系统构成或后台机柜时才画出。
- 概念图可用小型隐藏设备区、维护门、机柜轮廓或简短标注暗示后台。
- 不要把展项交互图画成弱电系统图或设备清单。

## 载体选择快速判断

- 明亮空间、大屏信息：优先 LED、LCD 商显、灯箱、实体模型，不优先普通投影。
- 暗场、大尺度沉浸、异形表面：投影/融合/纱幕/模型投影可行，但必须画出投影源和光路。
- 近距离儿童操作：低位触摸屏、实体按钮、道具识别、桌面屏、地面触发更合适。
- 地面踩踏和路径反馈：地踩 LED、压力传感、地面灯带、地面投影都可行，但不要放长文本。
- 实体操作转数字反馈：优先 RFID、磁吸、编码器、旋钮、拉杆、滑轨、机械开合，而不是只画屏幕 UI。
- 气味辨识和记忆内容：优先闻香口、样本托盘、气味胶囊或局部闻香器，搭配短标签/小屏结果，不优先大屏说明。
- 生物、生态、历史、城市类内容：屏幕/投影/LED 上必须显示主题相关信息层，不要显示通用蓝色界面。

## 硬件质检

生成后检查：

- 设备是不是像真实展陈硬件，而不是魔法发光物。
- 设备位置、安装方式、工作距离是否成立。
- 人会不会挡住投影、感应、观看或声音路径。
- 屏幕/LED/投影的亮度和空间光环境是否匹配。
- 气味释放口、嗅闻距离、通风和换样维护是否成立。
- 地面/桌面/墙面的内容颗粒度是否适合观看距离。
